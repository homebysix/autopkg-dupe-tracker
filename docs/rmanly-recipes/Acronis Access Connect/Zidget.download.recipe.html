Desciption Download recipe for the Acronis Dashboard widget called the Zidget. The HOSTNAME recipe variable is your local file or print server running Acronis Access Connect. You may already have a CNAME in DNS for ExtremeZIPServerList, the default. If you are still running ExtremeZ-IP this recipe works for that as well.
