Desciption Downloads the latest Oracle Java 8 JRE and builds a new package with correct version information. All Oracle JRE install packages have a version number of 1, so the actual plugin version is used to replace the installer version, taken from the CFBundleVersion of the bundle at /Library/Internet Plug-Ins/JavaAppletPlugin.plugin.
