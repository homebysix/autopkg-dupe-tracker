Desciption Downloads the latest release copy of GAM (Google Admin command line manager) from Github. Set PRERELEASE to a non-empty string to download prereleases, either via Input in an override or via the -k option, i.e.: `-k PRERELEASE=yes`. Available major macOS versions can be found at https://github.com/jay0lee/GAM/releases/latest.
