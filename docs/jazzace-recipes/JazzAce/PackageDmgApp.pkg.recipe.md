# PackageDmgApp.pkg.recipe

            _Last updated 2021-12-23 19:58:07Z_

            - **Description**: This is a generic recipe to turn any drag-and-drop app that is wrapped in a disk image into a .pkg installer.
The user must specify the path to the dmg at runtime via the --pkg option.
The package will be saved in the AutoPkg cache for this recipe.
A new package will be built each time.

            - **Identifier**: `com.github.jazzace.pkg.packagedmgapp`

            - **Parent Recipes**: `None`
