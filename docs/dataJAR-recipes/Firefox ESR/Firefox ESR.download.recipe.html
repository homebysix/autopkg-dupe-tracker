Desciption Downloads the latest Firefox ESR release by checking: https://product-details.mozilla.org/1.0/firefox_versions.json for the key: FIREFOX_ESR_NEXT if this exists, then this keys version is used to generate a download URL. If FIREFOX_ESR_NEXT does not exist, fails over to FIREFOX_ESR. Set the 'LOCALE' variable to the required language version. List of available locale codes here: https://ftp.mozilla.org/pub/firefox/releases/110.0.1/mac/ or here https://ftp.mozilla.org/pub/firefox/releases/latest/README.txt
