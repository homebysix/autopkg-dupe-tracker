# UnityAdditionalComponents.pkg.recipe

        _Last updated 2021-12-23 20:01:50Z_

        - **Description**: Downloads the latest version of the various unity assets. Note, this
recipe is designed to be used with an override, pointing to any of the
additional unity components listed here:

https://unity3d.com/unity/whats-new

The "%ADDITIONAL_COMPONENT%" variable supports one of the following:

Android
AppleTV
iOS
Linux-Mono
Lumin
Mac-IL2CPP
WebGL
Windows-Mono

        - **Parent Recipes**: `com.github.moofit-recipes.download.UnityAdditionalComponents`
