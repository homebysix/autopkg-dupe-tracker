Desciption Verifies your pre-downloaded Zend Studio download and prepares it for processing by child recipes. Zend Studio can be downloaded here: https://www.zend.com/downloads/zend-studio An email address is required to initiate the download. Once you've downloaded, pass the resulting file in to this recipe using the PKG variable or -p flag: autopkg run -v ZendStudio.download -p ~/Downloads/ZendStudio-13.6.1-macosx.cocoa.x86_64.dmg
