Desciption Downloads the latest version of Pano2VR and creates a package that deploys both the app and some preferred settings, particularly the license string. Details on how to obtain the license string are available in the FAQ (https://ggnome.com/faqs/?Display_FAQ=1900). The value of VERSION_CHECK and WELCOME_SCREEN should be set to either true or false (the string) depending on your preference.
