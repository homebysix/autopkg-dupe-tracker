Desciption Downloads the latest version of Fission. NOTE: This recipe used to use the vendor's CloudFront CDN download URL (https://d2oxtzozd38ts8.cloudfront.net/fission/download/Fission.zip) however AutoPkg users found that CloudFront was sometimes not kept updated as new versions were released. We are using the slower but more reliably updated rogueamoeba.com download link now.
