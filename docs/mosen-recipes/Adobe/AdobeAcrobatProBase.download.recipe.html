Desciption: Downloads latest Adobe Acrobat Pro base installer as provided at https://helpx.adobe.com/acrobat/kb/acrobat-dc-downloads.html MAJOR_VERSION can be either 'DC' or '2015'. DC should be the 'Continuous' track, '2015' is the current latest 'Classic' track. For more information on Continuous vs. Classic "tracks" see: http://www.adobe.com/devnet-docs/acrobatetk/tools/AdminGuide/whatsnewdc.html#which-track-should-i-use and http://www.adobe.com/devnet-docs/acrobatetk/tools/AdminGuide/whatsnewdc.html#preference-and-file-paths
