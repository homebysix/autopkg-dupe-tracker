Desciption Downloads the latest version of BeyondTrust Remote Support Jump Client from your appliance using the Remote Support API. You must override the following keys: * BEYOND_TRUST_URL * BEYOND_TRUST_CLIENT_ID * BEYOND_TRUST_CLIENT_SECRET * JUMP_GROUP * Can be the Group "name" or "ID" Optionally, you can override: * JUMP_CLIENT_PLATFORM (Processor supports this, but this recipe obviously does not) * JUMP_CLIENT_CONFIG * An example config is provided, but if it's not provided, the minimum will be used which will generated a Jump Client Key using the default values.
