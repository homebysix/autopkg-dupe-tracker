Desciption Downloads the current release of i1Profiler and extracts the installation package. This recipe is a stub, as the updated parent recipe provides a pkg installer. If it is your process to add versioning to the filename of the pkg, you may wish to write your own pkg recipe that leverages the fact that the version number is returned by the SparkleUpdateInfoProvider processor in the download recipe (%version%).
