Desciption This recipe downloads the signed installer package that Mozilla made available starting in Thunderbird 69.0, and imports the pkg file into a Munki repo. The RELEASE key used in the standard Thunderbird recipes are not yet supported. LOCALE controls the language localization to be downloaded. Examples include 'en-US', 'de', 'sv-SE', and 'zh-TW' See the following URL for possible LOCALE values: http://ftp.mozilla.org/pub/thunderbird/releases/latest/README.txt
