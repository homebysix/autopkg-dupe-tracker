# munkitools3-autobuild.munki.recipe

            _Last updated 2021-12-23 20:01:50Z_

            - **Description**: Note: munkitools does not include a code signature. If your
organization requires code signature, it is recommend to internally sign
the application.

Downloads and imports a pre-built Munki 3 from
https://munkibuilds.org. DOWNLOAD_URL can be overridden to
an alternate URL, from either your own server or an alternate
package version, ie.
https://munkibuilds.org/3.3.2.3541/munkitools-3.3.2.3541.pkg

MUNKI_ICON should be overridden with your icon name.
