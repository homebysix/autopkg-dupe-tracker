Desciption Downloads the latest Android Studio IDE from Google. For Apple Silicon set ARCH to "arm64" (default) and for Intel set ARCH to "x86_64". NOTE: This recipe differs from others in that it allows for the import and distribution of "patch" versions (not just major/minor) of Android Studio. It also allows for a single architecture variable to be used for both the download and Munki.
