Desciption Downloads the latest Intellij disk image and imports into Munki. Defaults to the Intellij Community Edition for intel architecture. Override the following keys for other versions: For macOS Silicon: - arch == "aarch64" For IntelliJ Ultimate Edition: - CODESIGNATUREAPPNAME == "IntelliJ IDEA.app" - CODESIGNATUREREQUIREMENT == "identifier "com.jetbrains.intellij" and anchor apple generic and certificate 1[field.1.2.840.113635.100.6.2.6] /* exists */ and certificate leaf[field.1.2.840.113635.100.6.1.13] /* exists */ and certificate leaf[subject.OU] = "2ZEFAR8TH3"" - edition == "U"
