Desciption Downloads the latest MacAdmins Python 3 framework pkg (recommended) and uploads it to the JSS. In order to get the latest pre-release, set the Input variable INCLUDE_PRERELEASES to a non-empty string (such as 'True'; this is the default). If you just want the latest full release instead, INCLUDE_PRERELEASES must be set to an empty string.
