Desciption Downloads the current release version of the JumpCloud Agent and then uploads to munki. NOTE: The bootstrap.json file should be deployed to /opt/jc/agentBootstrap.json before installing this PKG, else you'll be prompted for the connect_key. Amend the preinstall_script for this. See: https://support.jumpcloud.com/customer/portal/articles/2389320-agent-deployment-via-command-line Set the DERIVE_MIN_OS variable to a non-empty string to set the minimum_os_version via MunkiInstallsItemsCreator.
