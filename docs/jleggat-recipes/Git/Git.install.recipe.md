# Git.install.recipe

            _Last updated 2021-12-23 20:01:50Z_

            - **Description**: Downloads Git for OS X dmg from sourceforge and installs it.
By default downloads version for the newest OS X release.
Optionally specify MAJOR_OS_VERSION as "mavericks" or "snow-leopard" for compatability with OS X release.

            - **Identifier**: `com.github.jleggat.Git.install`

            - **Parent Recipes**: `com.github.jleggat.Git.download`
