CorelDRAWUpdate.munki.recipe

Last refresh: 2021-12-26 17:41:53

Desciption: Downloads the latest updates for CorelDRAW and imports it into Munki. The base version is modifiable via the BASE_VERSION input variable. The architecture can be chosen from Intel and M1 by modifying the ARCHITECTURE variable. Additionally the SUPPORTED_ARCHITECTURE variable has to be adjusted accordingly to x86_64 (Intel) or arm64 (M1)
