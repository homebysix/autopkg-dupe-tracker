Instead of setting the 'Processor' key to a processor name only, we separate the recipe identifier and the processor name with a slash: Processor com.github.autopkg.recipes.SampleSharedProcessor/SampleSharedProcessor ..assuming that this recipe is in one of AutoPkg's search dirs.
