Last refresh: 2021-12-26 17:14:49

Desciption: Downloads latest version of DaVinci Resolve 16 and turns it into a package. This recipe is a modification of moofit's original v16 pkg recipe. This product requires registration in order to download, so you _must_ override all input variables beginning with 'REG_' and they must contain data. Notably, REG_STATE and REG_COUNTRY must contain sane values: - REG_COUNTRY is of the form 'us', 'ca', etc. - REG_STATE is a proper name, i.e. 'Maryland'.
