ProToolsHDDriverImport.munki.recipe

Last refresh: 2021-12-26 17:41:52

Desciption: Copies the Pro Tools HD Driver out from a Pro Tools .dmg download and imports it into a munki_repo. To use this recipe: 1) Create a recipe override of this recipe. 2) Go to the Avid website and download the Pro Tools for Mac DMG. 3) Copy the downloaded DMG onto your Mac running autopkg in the folder specified in the PATH variable. Don't include a trailing '/'. 4) Run the recipe. 5) Profit.
