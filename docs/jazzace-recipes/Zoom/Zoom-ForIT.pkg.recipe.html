Desciption Downloads the latest Zoom installer for IT and adds the version number to the pkg name. You would choose this recipe over the pkg recipe from MLBZ521-recipes if you were not managing settings for Zoom at all OR if you were managing settings with a configuration profile. Note: Because Zoom has released more than one installer with the same version number in the past (e.g., 5.4.6), the build number is NOT stripped. The end result will look something like: ZoomInstallerIT-5.5.4 (13130.0228).pkg
