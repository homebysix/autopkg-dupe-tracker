Desciption Download the latest version of the Unity Windows Support pkg and imports it into a munki_repo. Need to supply the munki item name for Unity 3D in UNITY_MUNKI_ITEM_NAME to correctly link this support installer to it along with the correct version. A preinstall script in this support package checks to make sure the same version of Unity 3D is installed, otherwise it fails.
