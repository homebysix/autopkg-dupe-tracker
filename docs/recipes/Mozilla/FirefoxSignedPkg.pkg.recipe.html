Desciption This recipe downloads the signed installer package that Mozilla made available starting in Firefox 69.0. Because the downloaded file is already a package, this pkg recipe does not add any further actions. The RELEASE key used in the standard Firefox recipes are not yet supported. LOCALE controls the language localization to be downloaded. Examples include 'en-US', 'de', 'sv-SE', and 'zh-TW' See the following URL for possible LOCALE values: http://ftp.mozilla.org/pub/firefox/releases/latest/README.txt
