Desciption Downloads the latest version of OpenVPN Connect Client 3 and imports it into Munki. Set the ARCH variable to "x86_64" for Intel downloads or "arm64" for Apple Silicon downloads. Set the DERIVE_MIN_OS variable to a non-empty string to set the minimum_os_version via MunkiInstallsItemsCreator.
