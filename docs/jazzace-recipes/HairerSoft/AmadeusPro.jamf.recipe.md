# AmadeusPro.jamf.recipe

            _Last updated 2021-12-23 19:58:07Z_

            - **Description**: Downloads the latest version of Amadeus Pro, packages it, then uploads the package to the Jamf Pro Server 
and creates/updates the policy that deploys it. The POLICY_CATEGORY must exist on the Jamf Pro Server and the 
scope must be specified in the POLICY_TEMPLATE.
