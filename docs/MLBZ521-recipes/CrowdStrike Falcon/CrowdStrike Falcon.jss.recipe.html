Desciption Downloads a CrowdStrike Falcon Sensor pkg via CrowdStrike's API, creates a installation package and then uploads it to a JPS. You must provide a Client ID and Secret along with the Policy ID to determine which senor version to download. Expects your Customer ID (license ID) so that the pkg recipe can license the Sensor on install.
