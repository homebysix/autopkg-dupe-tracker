Desciption Downloads the latest TextMate 2 from their GitHub repo and imports into Munki. The postinstall script symlinks the 'mate' command-line helper tool from the application bundle to /usr/local/bin/mate, the same path that TextMate2 would install it to. Set PRERELEASE to a non-empty string to download prereleases, either via Input in an override or via the -k option, i.e.: `-k PRERELEASE=yes`
