Desciption Build an unattended version of X-Rite i1Profiler. Remove interactive scripts and applications (usually AppleScripts) to allow for unattended installation. Normally the i1Profiler mpkg runs loginitem.app to get i1ProfilerTray.app into Login Items. It also launches the tray application during the install. These have been removed as they fail in a non-interactive installation. OVERRIDES NAME - The name used to find the downloaded package that was unpacked from a zip file. This should normally not change. PKG_NAME - The name of the package that will be created by this recipe. Defaults to `i1Profiler Silent.pkg`. You may have to override this in i1Profiler.munki because FlatPkgPacker does not have output variables.
