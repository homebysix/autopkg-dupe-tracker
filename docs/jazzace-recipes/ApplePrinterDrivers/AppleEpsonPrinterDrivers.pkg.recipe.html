Note: Because this package is large (over 1 GB), the recipe suppresses unnecessary copying of the package using the "StopProcessingIf" processor. This means that any recipe that uses this as a parent recipe could stop execution before reaching the child. You may need to remove the cache for the recipe in question and start again if you manually delete or move the package that this recipe creates.
