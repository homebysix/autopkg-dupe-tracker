Desciption Downloads the current release version of Lightwright and then packages the Lightwright app. This recipe is based on the Version 6 app (now wrapped in a dmg). Note: The version number is included in the app name as of 6.0.51. This recipe renames the app bundle to the value of NAME. To be consistent with earlier versions (and to overwrite them upon deployment), set NAME to Lightwright 6.
