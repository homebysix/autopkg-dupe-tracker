Desciption: "Downloads" Rift, packages it for installation under After Effects CC 2020, and imports it into Munki. NOTE: - This recipe cannot determine the version of Rift and must have it fed in as Input, whether via override or via CLI with -k - This recipe does not download the Rift zip archive--feed the zip archive into the recipe via -p The sum of the two notes above would lead to an example run of this recipe as follows: autopkg run RiftAfterEffectsCC2020.munki -p /path/to/rift_v1.4.2.zip -k VERSION=1.4.2
