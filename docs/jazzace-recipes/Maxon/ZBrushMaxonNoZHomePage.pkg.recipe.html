Desciption This recipe takes the ZBrush installer app from Maxon and creates a package that leverages the silent install CLI method to automate installation. This also replaces the DefaultZScript.txt file in order to suppress the automatic launching of ZHomePage when opening ZBrush. You may set a different Bundle ID for your package by changing the BUNDLE_ID Input variable. This recipe requires version 2024.1.0 or newer (of ZBrush and the Maxon App).
