DaVinciResolveLite11.download.recipe

Last refresh: 2021-12-27 00:09:11

Desciption: Downloads latest version of DaVinci Resolve Lite 11. This product requires registration in order to download, so you _must_ override all input variables beginning with 'REG_' and they must contain data. Notably, REG_STATE and REG_COUNTRY must contain sane values: - REG_COUNTRY is of the form 'us', 'ca', etc. - REG_STATE is a proper name, i.e. 'Maryland'.
