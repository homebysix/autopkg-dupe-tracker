Desciption Downloads the latest version of the Epson SureColor P-series product driver from Epson's site. These will generally be drivers and ICC profiles not included in the OS by Apple. Make separate overrides for each printer you may want. An override is required. When Epson was using separate code signing authorities for older and new products, it was necessary to have separate recipes (EpsonProDrivers and EpsonSureColorDrivers). Now that Epson is using the same authority for all new releases, this recipe has been turned into a stub recipe that points to EpsonProDrivers.download. If you are creating a new override, use the EpsonProDrivers series of recipes.
