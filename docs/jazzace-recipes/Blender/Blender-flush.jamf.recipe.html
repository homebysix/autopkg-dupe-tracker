Desciption This recipe downloads Blender (with the architecture specified by ARCHITECTURE), packages it, then uploads the pkg to your Jamf Pro Server/Distribution Point, creates/updates the related policy, and flushes the Policy Log to deploy it. Because Blender has released separate installers for Intel and Apple Silicon, you will need to override this recipe for each architecture you support. The server/share details must be set using variables set in the environment or specified in your override (see https://github.com/grahampugh/jamf-upload/wiki/JamfUploader-AutoPkg-Processors). The grahampugh-recipes and hjuutilainen-recipes repos are required. See Blender-latest.jamf for details on the necessary Input variables.
