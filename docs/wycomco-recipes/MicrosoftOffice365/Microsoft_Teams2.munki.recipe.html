Desciption Downloads the latest Microsoft Teams installer package, appends the version to the end of the filename, creates a proper installs array and imports into Munki. This recipe is a fork of com.github.autopkg.munki.MSTeams. You may also want to change the attributeSetting for com.microsoft.autoupdate in installer_choices_xml to "0" to prevent Microsoft's own updater to be installed (line 63 in this recipe).
