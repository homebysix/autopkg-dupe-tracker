Audacity.munki.recipe

Last refresh: 2021-12-25 04:50:28

Desciption: Imports a manually-download copy of Audacity into Munki. This recipe originally provided an automated download of Audacity from fosshub.com, using a download parent recipe. Scripted scraping of fosshub.com is against their terms of service, and so this recipe requires a manual download of the .dmg and for it to be passed manually using the `--pkg` argument to `autopkg run`. Note: the Audacity.app code object is not signed at all.
