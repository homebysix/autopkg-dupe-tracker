Desciption Downloads and packages the latest version of BeyondTrust Remote Support Jump Client from your appliance using the Remote Support API. At this time, the created .pkg simply stages the Jump Client installer in /tmp as I install the client via a customized process via a script in Jamf Pro. * Script is here: https://github.com/MLBZ521/MacAdmin/blob/master/Software/Beyond%20Trust%20Bomgar/Install-BomgarJumpClient.py If you would like to see the recipe create a package that will actually install the client, submit feedback and I'll considering adding a postinstall script to this recipe. You must override the following keys: * BEYOND_TRUST_URL * BEYOND_TRUST_CLIENT_ID * BEYOND_TRUST_CLIENT_SECRET * JUMP_GROUP * Can be the Group "name" or "ID" Optionally, you can override: * JUMP_CLIENT_PLATFORM (Processor supports this, but this recipe obviously does not) * JUMP_CLIENT_CONFIG * An example config is provided, but if it's not provided, the minimum will be used which will generated a Jump Client Key using the default values.
