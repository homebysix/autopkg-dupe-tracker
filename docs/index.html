AutoPkg Recipe Deduplication

Recommendations for consolidating redundant recipes across the AutoPkg org
Maintain a recipe repo? See how you can help

How recipe maintainers can help:

  • Filter by your repo using the dropdown to find sets that affect your recipes.
  • Merge open deprecation PRs submitted to your repo.
  • Add DeprecationWarning to recipes flagged for deprecation if no PR exists.
  • Update ParentRecipe references to point to the recommended keeper.

Questions? Open an issue.

Recommendations are AI-assisted and intended as a basis for human review. Always read the actual recipes before acting on any recommendation.
