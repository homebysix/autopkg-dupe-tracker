AmazonCorretto.jss.recipe

Last refresh: 2021-12-26 01:38:07

Desciption: Downloads the Amazon Corretto OpenJDK, builds a package, and uploads it to the JSS. Optionally, this package will pull out the JVM Version which can be used with a Smart Group pointing to an extension attribute to determine latest version. Use the EXTENSION_ATTRIBUTE override variable to use your current EA. The JDK Major Version can be specified using the override variable "JDK_MAJOR_VERSION". Currently available from Amazon are the 8 and 11 JDKs.
