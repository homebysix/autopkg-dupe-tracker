Desciption Stub Recipe to locate the shared processors in this folder. To use the processors here, instead of setting the 'Processor' key to a processor name only, we separate the recipe identifier and the processor name with a slash: Processor com.github.scriptingosx.SharedProcessors/ProcessorName Currently ScriptingOSX has the following shared processors: - Notification: shows a OS X notification (Sample: XQuartz/XQuartz.notify.recipe) - SendPKGWithARD: uses ARD to send and install a pkg to a given computer list in Apple Remote Desktop - FileTemplate: copies a template file and fills placeholders with autopkg key (e.g. %NAME%, %version%) (used in FirefoxUSC.pkg)
