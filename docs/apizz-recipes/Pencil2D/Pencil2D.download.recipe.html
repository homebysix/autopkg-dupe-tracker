Desciption Downloads the latest release copy of Pencil2D from Github. Set PRERELEASE to a non-empty string to download prereleases, either via Input in an override or via the -k option, i.e.: `-k PRERELEASE=yes` 10.12+ version regex: ^pencil2d-mac-[\d.]+.zip$ 10.7-10.11 version regex: ^pencil2d-mac-legacy-[\d.]+.zip$
