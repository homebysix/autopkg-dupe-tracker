Desciption Downloads and packages the latest LibreOffice with the given language. Set RELEASE to either 'Latest' or 'Previous' and ARCH to either 'x86_64' or 'aarch64'. Please assign the appopriate LANGUAGE_CODE. This can be found by going to https://www.libreoffice.org/download/download-libreoffice, selecting "need another language", and selecting the language. The LANGUAGE_CODE can be found at the end of the URL after 'lang='
