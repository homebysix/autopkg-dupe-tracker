Desciption Downloads the latest version of Elgato Stream Deck and imports it into Munki. Set the DERIVE_MIN_OS variable to a non-empty string to set the minimum_os_version via MunkiInstallsItemsCreator. This requires a minimum AutoPkg version of 2.7 please update if you're not already running it. To force MunkiImporter to import the pkg, set FORCE_MUNKIIMPORT to a non-empty string.
