Desciption Downloads the latest release of customdisplayprofiles for the architecture specified by ARCHITECTURE (intel or applesilicon) and then creates a package with it and other tools stored locally: - a complied version of brightness (github.com/nriley/brightness); - login-every scripts for Outset (github.com/macadmins/outset, installed separately) to set the calibration profile (iaml_configure_display_profiles.sh, located at OUTSET_PROFILES_PATH) and brightness (SetCalibratedBrightness.sh, located at OUTSET_BRIGHTNESS_PATH); - Drop Calibration Profile Here app (located at DROPLET_PATH) that is used to set the profile and brightness after calibration via the GUI. I have hardcoded the target filenames for everything except customdisplayprofiles. If you want to do something similar but instead use the filename of the file, see look inside the JazzAce folder in my jazzace-recipes repo for an example of using StringSplitter to grab the filename from the pathname.
