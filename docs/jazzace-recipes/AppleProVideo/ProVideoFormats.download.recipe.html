Desciption Downloads the latest update of the Pro Video Formats provided by Apple for their pro apps. Note 1: This recipe requires the n8felton-recipes repo. Note 2: The downloaded installer is an updater that requires that you have one of the following installed: * Final Cut Pro X * Compressor 4.1 * Motion 4.0 or later Having that software installed means you also agreed to the appropriate Software License Agreement(s). This recipe supports Version 2.1 and later (requires macOS 10.14.3 or later).
