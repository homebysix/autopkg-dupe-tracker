Desciption This recipe takes the PTGui app (contained in a disk image that you have already downloaded) and creates an installer package for the app plus the JSON file you created containing the license keys for activation (as per https://ptgui.com/support.html#2_10). LIC_FILE is the path of the JSON file (this will be installed in the location required by PTGui). You should set this value in an override or at runtime using the -k flag. Your command to create the pkg will look something like this: autopkg run PTGuiVolume.pkg -p ~/Downloads/PTGui_12.11.dmg (Substitute the actual path to your downloaded disk image.)
