Desciption Downloads the latest version of VeraCrypt and imports it into Munki. Set the DERIVE_MIN_OS variable to a non-empty string to set the minimum_os_version via MunkiInstallsItemsCreator. This requires a minimum AutoPkg version of 2.7 please update if you're not already running it. Vera Crypt requires that macFUSE (previously osxfuse) to be installed. Recipes for this can be found here: https://github.com/autopkg/apizz-recipes/tree/master/macFUSE
