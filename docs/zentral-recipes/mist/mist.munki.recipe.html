Desciption Downloads and imports to Munki the latest version of Mist, a Mac utility that automatically downloads macOS Firmwares / Installers. Set the DERIVE_MIN_OS variable to a non-empty string to set the minimum_os_version via MunkiInstallsItemsCreator. This requires a minimum AutoPkg version of 2.7 please update if you're not already running it.
