Desciption Downloads latest version of Hopper Disassembler 4. Currently this downloads only a demo version. In order to download the full version, an additional query string, 'OrderIDHash', must be added to the appcast URL query. Unfortunately I'm not sure how to easily extract this from an existing license installation besides actually capturing the web traffic. When Hopper detects a valid license with the demo version, it will prompt and check for updates using the OrderIDHash so that it downloads the latest full version.
