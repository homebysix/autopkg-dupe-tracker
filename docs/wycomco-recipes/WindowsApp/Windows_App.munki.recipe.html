Desciption Downloads the latest version of Microsoft's Windows app installer package and imports into Munki. This recipe supports the automatic identification of the minimum_os_version and creates a proper installs array. You may also want to change the attributeSetting for com.microsoft.package.Microsoft_AutoUpdate.app in installer_choices_xml to "0" to prevent Microsoft's own updater to be installed (line 47 in this recipe).
