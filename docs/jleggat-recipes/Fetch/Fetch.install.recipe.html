Fetch.install.recipe

Last refresh: 2021-12-26 17:14:49

Desciption: Downloads the latest Fetch from Fetch Softworks website, creates a Apple installer pkg and then installs it. Optional Values SERIALNUMBER and REGISTRANTNAME when both provided will add a fetch license agreement to /Library/Preferences/com.fetchsoftworks.Fetch.License.plist. Set SERIALNUMBER to serial number provided in a Fetch license agreement. Set REGISTRANTNAME to name used to register a Fetch license agreement.
